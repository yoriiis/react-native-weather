## Available Scripts

In the project directory, you can run:

### `yarn start`

Runs the Javascript server.

### `yarn android`

Runs the app in the Android emulator (need Android Studio).

### `yarn ios`

Runs the app in the iOS emulator (need XCode).

### `yarn lint`

Runs ESLint on source files.
